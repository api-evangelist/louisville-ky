# Louisville, KYThis is a repo for managing machine readable artifacts for the [Louisville, KY APIs](http://portal.louisvilleky.gov/service/data). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Louisville, KY APIs](http://portal.louisvilleky.gov/service/data).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://portal.louisvilleky.gov/service/data)- [Developer](http://portal.louisvilleky.gov/service/data)The [APIs.json](https://github.com/api-evangelist/louisville-ky/blob/master/apis.json) index provides links to all the relevant links for the Louisville, KY APIs, including the OpenAPI and Postman if they are present.
